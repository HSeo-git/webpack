## 목적
webpack boilerplate
- dev mode
- prod mode
## 실행방법
```
[build]
prod 모드 빌드: npm run build
dev 모드 빌드: npm run build:dev
[start]
prod 모드 실행: npm run start
dev 모드 실행: npm run start:dev
[server 실행]
npm run server
```
